# 窗口分屏

## 分屏方式

```
:split  缩写 :sp 上下分屏 
:vsplit 缩写 :vs 左右分屏
```

## 窗口跳转

```
Ctrl-w w 激活下一个窗口
Ctrl-w j 激活下方窗口
Ctrl-w k 激活上方窗口
Ctrl-w h 激活左侧窗口
Ctrl-w l 激活右侧窗口
```

## 屏幕缩放

```
Ctrl-w = 平均窗口尺寸
```
